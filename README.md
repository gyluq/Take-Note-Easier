# 笔记工具
适用于笔记软件mybase8.0的记笔记工具，可边看视频边记笔记
- 背景透明、可固定在桌面
- F3截图，截图时隐藏窗体
- 检测剪切板功能，支持文本和单张图片(图片会转换为base64格式)

# 环境
| 环境      | 版本 |
| :---        |    :----:   |
| Python      | 3.9.13       |
| pywin32   | 306        |
| system-hotkey   | 1.0.3        |
| pyside6   | 6.5       |
